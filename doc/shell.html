Properties:

  • Performance heavily depends on the used gap sequence
  • Worst-case performace: О(n 2) / O(n log 2 n) - worst / best known sequence
  • Best-case performance: O(n log n)
  • Worst-case space complexity: О(n) total anf O(1) auxiliary

Code:

n = len(A)
gaps = genGapSequence(n)

for gap in gaps:
  for i in range(gap):
    for j in range(i+gap, n, gap):
      k = j
      while k > i and A[k] < A[k-gap]:
        A[k], A[k-gap] = A[k-gap], A[k]
        k -= gap
