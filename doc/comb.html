Properties:

  • Worst-case performance: О(n 2)
  • Best-case performance: О(n log n)
  • Average performance: Ω(n 2 / 2 p), where p is the number of increments
  • Worst-case space complexity: О(n) total and O(1) auxiliary

Code:

n = len(A)
k = 1.3
gap = floor(n / k)

while gap > 1:
  for i in range(n-gap):
    if A[i] > A[i+gap]:
      A[i], A[i+gap] = A[i+gap], A[i]
  gap = floor(gap / k)

sortBubble(A)
