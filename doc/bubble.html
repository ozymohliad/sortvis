Properties:

  • Worst-case performance: О(n 2) comparisons and swaps
  • Best-case performance: О(n) comparisons and O(1) swaps
  • Average performance: О(n 2) comparisons and swaps
  • Worst-case space complexity: О(n) total and O(1) auxiliary

Code:

n = len(A)
while True:
  swapped = False
  for i in range(n - 1):
    if (A[i] > A[i+1]):
      A[i], A[i+1] = A[i+1], A[i]
      swapped = True
  if not swapped:
    break
  n -= 1
