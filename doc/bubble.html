Властивості:

  • Найгірша швидкодія: О(n 2) порівнянь і обмінів
  • Найкраща швидкодія: О(n) порівнянь і O(1) обмінів
  • Середня швидкодія: О(n 2) порівнянь і обмінів
  • Найгірша просторова складність: О(n) загальна і O(1) додаткова

Код:

length = len(A)
while True:
  swapped = False
  for i in range(length - 1):
    if (A[i] > A[i+1]):
      A[i], A[i+1] = A[i+1], A[i]
      swapped = True
  if not swapped:
    break
  length -= 1
